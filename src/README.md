# react-bpmn-modeler

React bpmn modeler (Beta)

## Technologic Stack

- [React](https://es.reactjs.org/)
- [Docker](https://docs.docker.com/)
- [bpmn-js](https://github.com/bpmn-io/bpmn-js)

## Deploy with Docker

```bash
yarn run docker:build
yarn run docker:run
```

Stop app

```bash
yarn run docker:stop
```

## Run in dev mode

- Install Docker

- Run

```bash
yarn install
```

```bash
yarn install
```
